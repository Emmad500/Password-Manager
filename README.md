# PassEM - Secure Password Manager

PassEM is a secure, modern password manager designed to help you store and manage your credentials safely. Built with the MERN stack, it offers a clean, responsive interface and robust backend security.

## Features

- **Secure Storage:** Your passwords are stored safely in a MongoDB database.
- **Responsive Design:** Works perfectly on desktop, tablet, and mobile devices.
- **Easy Management:** Add, edit, and delete passwords with a single click.
- **Quick Copy:** One-tap copy feature for usernames and passwords.
- **Modern UI:** Built with a premium Zinc & Violet theme for a professional look.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB Atlas

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Emmad500/Password-Manager.git
    ```

2.  **Install dependencies:**
    ```bash
    npm run install-all
    ```

3.  **Set up Environment Variables:**
    Create a `.env` file in the `server` folder and add your MongoDB URI:
    ```env
    MONGODB_URI=your_mongodb_connection_string
    ```

4.  **Run the App:**
    ```bash
    npm start
    ```
    The app will open at `http://localhost:5173`.
